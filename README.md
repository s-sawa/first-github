# 課題 -チーズアカデミー-
​
## ①課題内容（どんな作品か）
- チーズの学校のLPサイト
​
## ②工夫した点・こだわった点
- ボタンのホバー、マウスオーバーでカーソル変更
- ボタン押下でアラート表示（Javascriptのonclickイベント)
- トップページに戻るボタン追加
- Image Creator(Microsoft Bing)で生成した画像を使用した点

​
## ③難しかった点・次回トライしたいこと(又は機能)

- 下部のフォームを作成する方法
- より多くのアニメーションを使いたかった
- 画像の角を丸くした
- 余裕があればチーズアカデミーではなくオリジナルページも作りたかった
